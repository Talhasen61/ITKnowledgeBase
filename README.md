# 📚 IT Knowledge Base

A modern IT Knowledge Base application developed with **ASP.NET Core MVC**, **Entity Framework Core**, and **PostgreSQL**.

## 🚀 Features

- 📂 Category Management (CRUD)
- 📝 Article Management (CRUD)
- 🔗 One-to-Many relationship between Categories and Articles
- 🔍 Article details page
- ✏️ Edit existing articles and categories
- 🗑️ Delete confirmation pages
- 🎨 Responsive Bootstrap 5 interface
- 🗄️ PostgreSQL database integration

## 🛠️ Technologies

- ASP.NET Core MVC
- Entity Framework Core
- PostgreSQL
- Bootstrap 5
- C#
- HTML5
- CSS3

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Talhasen61/ITKnowledgeBase.git
```

2. Open the project with Visual Studio.

3. Configure your PostgreSQL connection string in `appsettings.json`.

4. Apply migrations

```bash
Update-Database
```

5. Run the project.
